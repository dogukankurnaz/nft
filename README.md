### Description

Generates random pixel avatar. You can use it in your NFT drawings.

### Requirements

* Python 3+
* pip
* pagan

---------------------------------------

### Installation

Pagan

```
pip install pagan 
```

start nftgenerator.py file

```
python3 nftgenerator.py
```
